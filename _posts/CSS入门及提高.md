# 字体设置

## 字体综合设置

```css
选择器 {font: font-style font-weight font-size/line-weight font-family;}

```

注意：

- 顺序不能变.
  
- 不需要设置的属性可以省略，取默认值，但必须保留 font-size 与 font-family

## 选择器

### 标签选择器（元素选择器）

```css
标签名 { 属性1:属性值1;属性2：属性值2}
```

### 类选择器

```css
.类名 {属性1:属性值1;属性2：属性值2}
```
